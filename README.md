# MindCapsule — Hug the Hedgehog

An AI-assisted emotional support system designed for adolescents experiencing academic stress. Combining Natural Language Processing (NLP) and positive psychology, this project provides a private space for emotional expression, recognition, and personalized support.

## Overview：
"Hug the Hedgehog" is an interdisciplinary project combining artificial intelligence, human-computer interaction, and psychology.
The project focuses on a common problem among high school students: many experience academic anxiety but hesitate to express their emotions because of social pressure and emotional stigma.
By analyzing users' text input, the system identifies possible emotional states and provides personalized psychological guidance to help users understand and regulate their emotions.

## Key Features：
- Text-based emotional interaction
- NLP-based emotion recognition
- Personalized psychological feedback
- Privacy-focused local deployment

## Technical Architecture：
User Input  -  Text Processing  -  Emotion Recognition  -  Psychological Support Generation  -  Personalized Feedback
